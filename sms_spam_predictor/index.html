Text Preprocessing Pipeline

Raw text is noisy. Our NLTK stemming pipeline applies the following normalization stages:

  1. Lowercasing: Removes case sensitivity issues.
  2. Link/HTML Stripping: Removes links and standard HTML tags.
  3. Punctuation Clean-up: Keeps alphabetic characters only.
  4. Stopwords Filtering: Eliminates semantic fillers (e.g., "and", "the").
  5. Porter Stemming: Reduces words to roots (e.g., "winner" -> "win").
